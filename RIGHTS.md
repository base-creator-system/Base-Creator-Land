# BASE Rights and Ownership

## Purpose

This document defines the rights boundaries between the BASE Framework and individual Creator Deployments built with BASE.

## Definitions

**BASE Framework** means the BASE software system, architecture, Creator Dashboard, Fan App structure, deployment workflow, and framework-level documentation.

**BASE Brand** means the BASE name, BASE logo, BASE identity, BASE marks, and public-facing identity connected to the BASE Framework.

**Creator Deployment** means a specific implementation of the BASE Framework for a creator, artist, business, organization, project, or brand.

**Creator Identity** means the creator's name, stage name, logo, visual identity, business identity, public-facing identity, and creator-specific presentation.

**Creator Content** means creator-supplied photos, videos, audio, music, text, artwork, links, live information, community links, supporter information, and media.

**Creator Audience** means the creator's direct relationships, supporters, customers, fans, email contacts, community relationships, and audience connections.

## Reserved Framework Rights

BASE reserves all rights in:

- BASE Framework
- BASE Architecture
- BASE software structure
- BASE Creator Dashboard system
- BASE Fan App system
- BASE framework documentation
- BASE framework development
- Future BASE framework versions
- BASE Brand
- BASE name, logo, identity, and marks

## Creator Rights

Each creator retains rights to:

- Creator Identity
- Creator Content
- Creator Media Library
- Creator Audience
- Creator business identity
- Creator logos
- Creator artwork
- Creator names
- Creator public-facing presentation
- Creator-supplied links
- Creator descriptions
- Creator communications

## Separation

A Creator Deployment is built with BASE.

BASE provides the framework.

The creator provides the Creator Identity, Creator Content, Creator Media Library, and Creator Audience.

Using BASE does not transfer Creator rights to BASE.

Using BASE does not transfer BASE Framework rights to the creator.

## Future Evolution

BASE may evolve through future framework versions, licensing models, commercial terms, governance structures, and written agreements.

Nothing in this document transfers BASE Framework rights or Creator rights unless expressly stated in a separate written agreement.

## Legacy Intent

BASE is intended to preserve long-term ownership, creative control, and future opportunity for its creator, successors, family, and authorized assigns.

No informal use, test, pilot, deployment, setup, or demonstration shall be interpreted as transferring ownership of the BASE Framework, BASE Brand, or future BASE Framework development rights.
